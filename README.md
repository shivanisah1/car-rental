# Car Rental App 🚗

Welcome to the Car Rental project! This is modern web application platform for browsing, booking, and managing car rentals.

## Project Structure

- **`/client`**: The frontend React app built with Vite, utilizing an ultra-modern dynamic design powered by Tailwind CSS.
- **`/my-projectcd`**: Additional project assets and components.

## Getting Started

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Running the Frontend

1. Navigate to the frontend directly:
   ```bash
   cd client
   ```
2. Install the necessary dependencies:
   ```bash
   npm install
   ```
3. Boot up the local development server:
   ```bash
   npm run dev
   ```
4. Check out the app at `http://localhost:5173` (or the port specified in your terminal).

## Technologies Used

- **React 19**
- **Vite**
- **Tailwind CSS v4**
- **React Router**
